Instructions

Input Page

  1. Click on "Choose File" and select the required file for text extraction.
  2. Accepted file types are - .jpg, .jpeg, .png, .pdf, .bmp, .gif, .tif, .tiff.
  3. Click on the "Submit" button to perform text extraction.
  4. Click on the "Reset" button to refresh the page.

Output Page

  1. The extracted text in editable format is available on the left-hand side of the screen.
  2. The original image is provided on the right-hand side of the screen.
  3. Click on "Download text as a file" to get the edited text in .txt format.
  4. Click on "Download OCR output as file" to get the raw text in .txt format.
  5. Click on the "Back" button to return to the homepage.
  6. Click on Raw Text in the bottom screen to view the extracted text in raw format
